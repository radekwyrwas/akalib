1. Overview

BondOAS® provides a self-consistent lattice-based recursive valuation kernel for fixed coupon bonds. The underlying interest rate model is the industry standard Black-Karasinski process implemented using AKA’s proprietary tetranomial lattice (or interest rate tree).

BondOAS® values all fixed coupon bonds, in terms of coupon structure (including zeros and step-ups), principal prepayment structure (bullets and amortizing), and optionality (calls, puts, sinking funds). These capabilities are applicable to all relevant market sectors (Corporates, Agencies, Treasuries, Municipals, domestic or foreign). All functions accept standard daycount conventions.

3. Programming Information

BondOAS® is written in C and is compiled for Win32, Solaris and Linux platforms. It is delivered as both static (akawin32.lib, libakasolaris.a, libakalinux.a) and dynamic (akadll.dll, akasolaris.so, akalinux.so) libraries. Information on how the library was compiled is in the file build.info in the architecture specific library directory.

BondOAS® is thread safe.

BondOAS® has a rich <a href=akaapi.html>application programming interface</a> which allows detailed control over valuation functionality.

An example directory is provided with the source in src/example. The directory contains a Makefile to compile the sample programs for the appropriate architecture.
